1.<p> 단락을 만들어 주는 태그

2. <b> 태그는 텍스트를 굵게 해줄때 사용합니다

3. < h>  태그는 제목 요소를 사용 할때 사용합니다

 <h1>–<h6> 요소는 6단계의 구획 제목을 나타냅니다. 구획 단계는 <h1>이 가장 높고 <h6>은 가장 낮다 

4.<title>  태그는 웹 페이지의 제목을 나타내는 태그입니다.

5.<body> 태그는 해당 HTML 문서의 텍스트, 하이퍼링크, 이미지, 리스트 등과 같은 모든 콘텐츠를 포함하는 영역을 정의할 때 사용합니다.

HTML 문서에는 단 하나의 <body> 요소만이 존재할 수 있습니다.

6.<ul> 태그는 순서가 없는 HTML 리스트(list)를 정의할 때 사용합니다.

<ul> 요소에 속하는 각 아이템은 <li> 요소를 사용할 수 있습니다

7. <li>태그는 list의 약자로, 목록을 만드는 태그입니다.
이 태그는 단독으로 쓰이지 않으며 <ul></li> 혹은 <ol></ol> 태그 내부에 들어갑니다.

<ol> 태그는 번호를 메겨 순서가 있는 (ordered list) 목록을 만듭니다.
<ul> 태그는 순서없이 모양으로 (unordered list) 목록을 만듭니다.
<li> 태그는 단순히 리스트 나열 뿐만 아니라 메뉴등을 만들때도 사용합니다.
이중 리스트를 만들기 위해서는 <ul>이나 <ol>하위에 다시 <ul>이나 <ol> 태그를 쓰면 됩니다.

8. <a>태그는 하이퍼링크를 걸어주는 태그입니다.

9. <img> 태그의 src 속성은 이미지 소스의 URL을 명시합니다.

10. alt 속성은 이미지를 보여줄 수 없을 때 해당 이미지를 대체할 텍스트를 명시합니다.

2일차 

1.  div 태그를 사용하여 각각의 블록(공간)을 알맞게 배치하고 CSS를 활용하여 스타일을 적용할 수 있습니다

2.<span> <span> 요소는 그 자체만으로는 어떠한 의미도 가지지 않지만, class나 id와 같은 전역 속성과 함께 사용하여 스타일링을 위해 요소들을 그룹화하거나 lang 속성과 같은 속성값을 공유하는 데 유용하게 사용할 수 있습니다.

3. div 태그와 span 차이점

 <div>는 줄 바꿈이 되지만, <span>은 줄 바꿈이 되지 않다는 점입니다.

4. <hr> 태그는 콘텐츠 내용에서 주제가 바뀔 때 사용할 수 있는 수평 가로선을 정의할 때 사용합니다.

5.<br> 줄바꿈 태그입니다

6. <sup> 태그는 윗첨자(superscript) 텍스트를 표현할 때 사용합니다.

7. 엔티 디코드 사이트가 있는데 특수문자 코드표 알려주는 사이트입니다

3일차 

table(테이블) 태그 : 데이터 표를 만드는 태그.
- <tr> 태그는 table row의 약자로 테이블의 row를 의미한다.
- td 태그 : 테이블의 열을 만드는 태그.
- th 태그 : 테이블의 열(타이틀-제목)을 만드는 태그.
- thead 테이블의 열의 머리글인 행들의 집합입니다.
- rowspan 하나의 셀을 두개 이상으로 확장하거나 셀을 여러 행으로 나눌 때 사용합니다.
